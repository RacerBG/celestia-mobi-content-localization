Celestia (Android/iOS/macOS) 1.5.20 and Celestia for UWP (Windows/Xbox) 2.0.36.0 are released.

App updates

  1. Added setting for tinted illumination saturation (Settings -> Star Styles, on Windows: Settings -> Renderer)
  2. Added setting for star colors (Settings -> Star Styles, on Windows: Settings -> Renderer)
  3. Added setting for game controller actions (Settings -> Game Controller)
  4. Added setting for custom flight mode (Camera Control -> Flight Mode, on Windows: Navigation -> Flight Mode)
  5. Added game controller navigation support for Xbox
  6. Show error message from core

Core updates

  1. Update galaxy and globular rendering
  2. Performance improvements
  3. Avoid crashing on bad models

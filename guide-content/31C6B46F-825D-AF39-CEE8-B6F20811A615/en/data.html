The original Celestia is a real-time space simulation that lets you experience our universe in three dimensions.
celestia.mobi hosts all the contents about Celestia frontends developed by Levin Li including an add-on collection maintained by the Celestia community. You can get donwloads of these frontends at Downloads.

Privacy

Celestia only collects data for analytic and diagnostic purposes and the code for collecting these data is publicly available for viewing. You can find the privacy policy required for Celestia here.

Development

Celestia is an open source software which anyone can contribute to. There is a guide for setting up for building Celestia here. Additionally, Celestia is built on top of third party libraries listed here.

Contacts

For questions and suggestions about Celestia in general, the original Celestia project has an online forum. There is also a discussion page on GitHub. You can also join the discord server.

For questions and suggestions about the frontends I maintain or about the website, please make a GitHub issue here or send an e-mail.
